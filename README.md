# Surprise-App-móvil

Este repositorio alberga el código fuente y los recursos necesarios para el desarrollo de la aplicación móvil **Surprise**, utilizando Flutter como framework multiplataforma y aplicando prácticas de desarrollo ágil.

El proyecto incluye pantallas principales como:
- Login
- Inventario
- Dashboard
- Reportes
- Promociones
- Pantalla de bienvenida

La estructura del proyecto sigue las recomendaciones oficiales de Flutter y se integra con un backend REST, utilizando autenticación JWT y manejo seguro de datos.
